CSV Import Tool

Imports a Google Sheets attendance export into Supabase ta_data. Merge-safe: other months are preserved, same-date rows are replaced, rid continues past the current max, and existing Settings are never overwritten.
Before you start: paste raw CSV (File → Download → CSV in Google Sheets, then open it in a text editor). Copying the rendered table from the Sheets web view produces tab/HTML content, not CSV — that is what causes "No data rows found".
