Rebuild Backup from Sheet Tabs

Reads your surviving month tabs and rebuilds a restore file for the app's ♻️ Restore button. Fully offline — this page never connects to anything and never writes to your Sheet.
Before you start: in the live app press 💾 Backup once and keep that file. Load it below as the base so your current month and your Settings are preserved exactly.

1 · Load your current backup (the base)

Optional but recommended — carries over Settings and any month already in the app.

2 · Easiest: drop the whole workbook

In the Sheet: File → Download → Microsoft Excel (.xlsx). One file, every tab. Drop it here and all month tabs are read at once — no per-month downloads.

2b · Or add month tabs one CSV at a time

In the Sheet: click the month tab → File → Download → Comma-separated values (.csv). Repeat per month. Add them one at a time below — combined tabs are fine, the SUBTOTAL / EARNINGS / GRAND TOTAL / LEAVE BALANCE rows are ignored automatically.

3 · Review and download the restore file

Nothing loaded yet.

4 · Restore in the app

Open the live app → ♻️ Restore → pick the downloaded file → confirm the row count. The app replaces its data, recalculates every row from Time In/Out and your Settings, then saves back to the Sheet automatically.

Check the row count in the confirm dialog before accepting. If it looks wrong, cancel — nothing changes.
